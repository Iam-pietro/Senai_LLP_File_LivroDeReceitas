# 📖 Livro de Receitas

Aplicação em Java que permite cadastrar receitas culinárias e salvá-las automaticamente em arquivos de texto.

---

## 📋 Descrição

O programa guia o usuário pelo cadastro de uma receita completa:

- **Nome** da receita
- **Ingredientes** (descrição + quantidade), um a um
- **Instruções** passo a passo

Ao finalizar, gera automaticamente um arquivo `.txt` com o nome da receita contendo todas as informações formatadas.
